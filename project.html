Digital Display

A wireless LED notice board is a system that allows users to send data to a display board wirelessly using a mobile phone. The system consists of a Wi-Fi Module, a LED P10 display , and a power supply. The Wi-Fi module receives the data from the mobile phone and sends them to the Wi-Fi module. The Wi-Fi module then displays the data on the P10 display.
